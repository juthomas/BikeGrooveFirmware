
## Pins Attribution
### Amplificators :
- Activate (High): `IO27`
- I2S :
  - bck (BCLK) : `IO27`
  - ws (LRCLK) : `IO26`
  - data (DIN) : `IO25`

### Switchs (Pull-up) :
 - Power SW1 : `IO33`
 - Down SW2  : `IO22`
 - Up SW4    : `IO23`

### Battery Probe :
- Voltage : `IO21` * 2

### Leds :
 - Blue : `IO19`
 - Red  : `IO18`
